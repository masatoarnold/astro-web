You are browsing through this project right now 😉

It is based on the [Astro Micro](https://astro-micro.vercel.app/) theme by Trevor Lee, which itself is based on [Astro Nano](https://astro-nano-demo.vercel.app/) by Mark Horn.

My website has inherited most of the convenience features: snappy performance, accessibility, a minimalist design, pagefind and Markdown support. I've stripped off Giscus (the comment system) and modified all the icons to use [Lucide](https://lucide.dev/). There are some other minor UI adjustments I made. I am honored to have such a strong foundation to build upon, and I would recommend anyone to check out those two great UI designers.
